# YouTube Clone with React and Redux

A YouTube clone built with React and Redux. Allows users to search and view YouTube videos.

## Features
 * Authentication with Firebase
  - SignUp,SignOut
 * Subscribe to a channel
 * View Subscribed Channels
 * View liked videos
 * View History videos
 * Search Videos

## Used Technologies
 * React.js
 * React-redux
 * Tailwind CSS
